Expected Total Profit

Definition. The value function is defined as $v^n(i)=\mathbb{E}[g(X_1)+\cdots+g(X_n)\mid X_0=i]$.

Interpretation. The value function is the total profit in $n$ days.

We may also calculate the total discounted profit $\mathbb{E}[\beta g(X_1)+\beta^2g(X_2)+\beta^3 g(X_3)\mid X_0=i]$. The intuition is that $100$ yuan on day $1$ may only be worth $95$ yuan on day $0$ if $\beta=0.95$.

Computing the Value Function Efficiently

A Naive Algorithm

We know that the value function is given by $(P+P^2+P^3)g$. We may compute its value in the following way.

Algorithm (Naive).

  1. Compute $P^2$
  2. Compute $P^3=P\cdot P^2$
  3. Sum $P+P^2+P^3$
  4. Compute $(P+P^2+P^3)g$

Suppose the number of states is $m=|S|$. The complexity of this algorithm is $O(2m^3)+O(m^2)=O(m^3)$.

Bellman Equation

We now introduce a more efficient algorithm which has complexity $O(m^2)$.

Define the cost-to-go function $v^k(j)=\mathbb{E}[g(X_k)+\cdots+g(X_3)\mid X_k=j]$, $k=0,1,2,3$. Then $v_0=v$.

Algorithm.

  1. Let $v^3=g$.
  2. Compute $v^2=g+Pv^3$.
  3. Compute $v^1=g+Pv^2$.
  4. Compute $v^0=g+Pv^1$.

This algorithm involves only matrix-vector multiplication and vector addition, which both have complexity $O(m^2)$. Thus the complexity of this algorithm is $O(m^2)$.

The rational behind this algorithm is the Bellman Equation.

Theorem. (Bellman Equation) $v^k=g+Pv^{k+1}$.

Proof.

Hence $v^k=g+Pv^{k+1}$. Q.E.D.

Bellman Equation: Infinite Horizon

If we are considering the expected total discounted profit over an infinite horizon

(note that a discount factor $\gamma\in(0,1)$ is necessary, for otherwise the sum would diverge), we have the following version of Bellman Equation.

Theorem. (Bellman Equation: Infinite Horizon) Assume that $g$ is bounded. Then

  • The value function satisfies $v=g+\gamma Pv$.
  • The solution $v$ exists and is unique.

This theorem holds even when $|S|=\infty$.

Proof of (a).

We conclude that $v=g+\gamma Pv$. Q.E.D.
