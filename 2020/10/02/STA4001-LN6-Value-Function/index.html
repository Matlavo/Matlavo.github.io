The Value Function

Expected Profit

Back to the Newsvendor model. Let g(Xn)g(X_n)g(Xn​) be the profit on day nnn. Then

Expected Total Profit

Definition. The value function is defined as vn(i)=E[g(X1)+⋯+g(Xn)∣X0=i]v^n(i)=\mathbb{E}[g(X_1)+\cdots+g(X_n)\mid X_0=i]vn(i)=E[g(X1​)+⋯+g(Xn​)∣X0​=i].

Interpretation. The value function is the total profit in nnn days.

The value function can be calculated in the following way:

(vn(1)vn(2)vn(3))=(P+P2+⋯+Pn)(g(1)g(2)g(3)) .\begin{pmatrix} v^n(1) \\ v^n(2) \\ v^n(3) \end{pmatrix} = (P+P^2+\cdots+P^n) \begin{pmatrix} g(1) \\ g(2) \\ g(3) \end{pmatrix}\,. ⎝⎜⎛​vn(1)vn(2)vn(3)​⎠⎟⎞​=(P+P2+⋯+Pn)⎝⎜⎛​g(1)g(2)g(3)​⎠⎟⎞​.

We may also calculate the total discounted profit E[βg(X1)+β2g(X2)+β3g(X3)∣X0=i]\mathbb{E}[\beta g(X_1)+\beta^2g(X_2)+\beta^3 g(X_3)\mid X_0=i]E[βg(X1​)+β2g(X2​)+β3g(X3​)∣X0​=i]. The intuition is that 100100100 yuan on day 111 may only be worth 959595 yuan on day 000 if β=0.95\beta=0.95β=0.95.

Computing the Value Function Efficiently

A Naive Algorithm

We know that the value function is given by (P+P2+P3)g(P+P^2+P^3)g(P+P2+P3)g. We may compute its value in the following way.

Algorithm (Naive).

  1. Compute P2P^2P2
  2. Compute P3=P⋅P2P^3=P\cdot P^2P3=P⋅P2
  3. Sum P+P2+P3P+P^2+P^3P+P2+P3
  4. Compute (P+P2+P3)g(P+P^2+P^3)g(P+P2+P3)g

Suppose the number of states is m=∣S∣m=|S|m=∣S∣. The complexity of this algorithm is O(2m3)+O(m2)=O(m3)O(2m^3)+O(m^2)=O(m^3)O(2m3)+O(m2)=O(m3).

Bellman Equation

We now introduce a more efficient algorithm which has complexity O(m2)O(m^2)O(m2).

Define the cost-to-go function vk(j)=E[g(Xk)+⋯+g(X3)∣Xk=j]v^k(j)=\mathbb{E}[g(X_k)+\cdots+g(X_3)\mid X_k=j]vk(j)=E[g(Xk​)+⋯+g(X3​)∣Xk​=j], k=0,1,2,3k=0,1,2,3k=0,1,2,3. Then v0=vv_0=vv0​=v.

Algorithm.

  1. Let v3=gv^3=gv3=g.
  2. Compute v2=g+Pv3v^2=g+Pv^3v2=g+Pv3.
  3. Compute v1=g+Pv2v^1=g+Pv^2v1=g+Pv2.
  4. Compute v0=g+Pv1v^0=g+Pv^1v0=g+Pv1.

This algorithm involves only matrix-vector multiplication and vector addition, which both have complexity O(m2)O(m^2)O(m2). Thus the complexity of this algorithm is O(m2)O(m^2)O(m2).

The rational behind this algorithm is the Bellman Equation.

Theorem. (Bellman Equation) vk=g+Pvk+1v^k=g+Pv^{k+1}vk=g+Pvk+1.

Proof.

\begin{align} v^k(i) &= \mathbb{E}[g(X_k)+g(X_{k+1})+\cdots+g(X_n)\mid X_k=i] \\ &= \mathbb{E}[g(X_k)\mid X_k=i] + \mathbb{E}[g(X_{k+1})+\cdots+g(X_n)\mid X_k=i] \\ &= g(i)+\sum_j\mathbb{E}[g(X_{k+1})+\cdots+g(X_n)\mid X_k=i,\,X_{k+1}=j]\mathbb{P}(X_{k+1}=j\mid X_k=i) \\ &= g(i)+\sum_j\mathbb{E}[g(X_{k+1})+\cdots+g(X_n)\mid X_{k+1}=j]\mathbb{P}(X_{k+1}=j\mid X_k=i)\tag{Markov} \\ &= g(i)+\sum_jv^{k+1}(j)P_{i,j} \\ &= g(i)+(Pv^{k+1})_i\,. \end{align}

Hence vk=g+Pvk+1v^k=g+Pv^{k+1}vk=g+Pvk+1. Q.E.D.

Remark. Here we have used the fact E[X]=EY[EX[X∣Y]]=∑yE[X∣Y=y]P(Y=y)\mathbb{E}[X]=\mathbb{E}_Y[\mathbb{E}_X[X\mid Y]]=\sum_y\mathbb{E}[X\mid Y=y]\mathbb{P}(Y=y)E[X]=EY​[EX​[X∣Y]]=∑y​E[X∣Y=y]P(Y=y). The proof is simple:

\begin{align} \mathbb{E}[X] &= \sum_{x}x\mathbb{P}(X=x) \\[4pt] &= \sum_xx\sum_y\mathbb{P}(X=x\mid Y=y)\mathbb{P}(Y=y) \\[4pt] &= \sum_y\sum_xx\mathbb{P}(X=x\mid Y=y)\mathbb{P}(Y=y) \\[4pt] &= \sum_y\mathbb{E}[X\mid Y=y]\mathbb{P}(Y=y)\,. \end{align}

More generally, we have E[X∣Z=z]=∑yE[X∣Y=y,Z=z]P(Y=y∣Z=z)\mathbb{E}[X\mid Z=z]=\sum_y\mathbb{E}[X\mid Y=y,Z=z]\mathbb{P}(Y=y\mid Z=z)E[X∣Z=z]=∑y​E[X∣Y=y,Z=z]P(Y=y∣Z=z).

Bellman Equation: Infinite Horizon

If we are considering the expected total discounted profit over an infinite horizon

v(i)=E[g(X0)+γg(X1)+γ2g(X2)+⋯∣X0=i]=E[∑n=0∞γng(Xn)∣X0=i]v(i)=\mathbb{E}[g(X_0)+\gamma g(X_1)+\gamma^2g(X_2)+\cdots\mid X_0=i]=\mathbb{E}\left[\sum_{n=0}^{\infty}\gamma^ng(X_n)\mid X_0=i\right] v(i)=E[g(X0​)+γg(X1​)+γ2g(X2​)+⋯∣X0​=i]=E[n=0∑∞​γng(Xn​)∣X0​=i]

(note that a discount factor γ∈(0,1)\gamma\in(0,1)γ∈(0,1) is necessary, for otherwise the sum would diverge), we have the following version of Bellman Equation.

Theorem. (Bellman Equation: Infinite Horizon) Assume that ggg is bounded. Then

  • The value function satisfies v=g+γPvv=g+\gamma Pvv=g+γPv.
  • The solution vvv exists and is unique.

This theorem holds even when ∣S∣=∞|S|=\infty∣S∣=∞.

Proof of (a).

\begin{align} v(i) &= \mathbb{E}\left[\sum_{n=0}^{\infty}\gamma^ng(X_n)\mid X_0=i\right] \\[4pt] &= g(i)+\mathbb{E}\left[\sum_{n=1}^{\infty}\gamma^ng(X_n)\mid X_0=i\right] \\[4pt] &= g(i)+\sum_{j}\left\{\mathbb{E}\left[\sum_{n=1}^{\infty}\gamma^ng(X_n)\mid X_0=i,X_1=j\right]\mathbb{P}[X_1=j\mid X_0=i]\right\} \\[4pt] &= g(i)+\sum_{j}\left\{\mathbb{E}\left[\sum_{n=1}^{\infty}\gamma^ng(X_n)\mid X_1=j\right]\mathbb{P}[X_1=j\mid X_0=i]\right\} \tag{Markov} \\[4pt] &= g(i)+\sum_{j}\gamma\left\{\mathbb{E}\left[\sum_{n=1}^{\infty}\gamma^{n-1}g(X_n)\mid X_1=j\right]P_{i,j}\right\} \tag{Markov} \\[4pt] &= g(i)+\gamma\sum_j v(j)P_{i,j} \\ &= (g+\gamma Pv)_i\,. \end{align}

We conclude that v=g+γPvv=g+\gamma Pvv=g+γPv. Q.E.D.
