Existence and Uniqueness

Theorem. (Bellman Equation: Infinite Horizon) Assume that $g$ is bounded. Then

  • The value function satisfies $v=g+\gamma Pv$.

Computing the Value Function

Solving a Linear System

To compute $v$, we may solve the linear system $(I-\gamma P)v=g$, which has complexity $O(m^3)$.

Value Iteration

Alternatively, we may use the following algorithm:

Algorithm.

  • Initialize $v^0$
  • Compute $v^k=f(v^{k-1})=g+\gamma Pv^{k-1}\,,\quad k=1,2,…$

Theorem. The iterates ${v^k}$ converges to $v^$ with $\Vert v^{k+1}-v^\Vert_2\le L^k\Vert v^1-v^0\Vert_2$.
