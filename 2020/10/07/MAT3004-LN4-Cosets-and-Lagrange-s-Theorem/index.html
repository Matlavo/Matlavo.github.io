Partition and Equivalence

Definition. Let $S$ be a nonempty set. A partition $P$ of $S$ is a collection $P={S_i\mid i\in I}$ of subsets of $S$ satisfying

  • $S_i\neq\empty$ $\forall\,i\in I$
  • $S_i\cap S_j=\empty$ $\forall\,i\neq j$
  • $\bigcup_{i\in I}S_i=S$

Definition. An equivalence relation on $S$ is a relation “$\sim$” satisfying

  • (Reflexive/Reflexivity) $a\sim a$ $\forall\,a\in S$
  • (Symmetric/Symmetry) $a\sim b$ implies $b\sim a$
  • (Transitive/Transitivity) $a\sim b$ and $b\sim c$ imply $a\sim c$

Theorem.

  1. If ${S_i\mid i\in I}$ is a partition of $S$, then the relation $\sim$ defined by “$a\sim b$ if and only if $a,b\in S_i$ for some $i$” is an equivalence relation.
  2. If $\sim$ is an equivalence relation on $S$, then ${C_a\mid a\in S}$, where $C_a={b\in S\mid b\sim a}$ is the equivalence class with representative $a$, is a partition of $S$.

Proof of 1. Reflexivity is obvious. Symmetry is also obvious. For transitivity, if $a\sim b$ and $b\sim c$, then $a,b\in S_i$ and $b,c\in S_j$. Then $b\in S_i\cap S_j$. Then $i=j$. So $a,c\in S_i$ and thus $a\sim c$. Q.E.D.

Proof of 2. $\forall\,a\in S$, since $a\sim a$, then $a\in Ca$, thus $C_a\neq\empty$ and $S=\bigcup{a\in S}C_a$.

To see that if $C_a\neq C_b$, then $C_a\cap C_b=\empty$, we will show that if $C_a\cap C_b\neq\empty$, then $C_a=C_b$. Then $\forall\,a,b\in S$, $C_a$ and $C_b$ are either the same set or disjoint. If $c\in C_a\cap C_b$ for some $c\in S$, then $c\sim a$ and $c\sim b$. Then $a\sim b$. Then for all $d$ such that $d\in C_b$, we have $d\sim b$ and thus $d\sim a$ and $d\in C_a$. Therefore $C_b\subset C_a$. Swapping $a$ and $b$ in the previous argument implies $C_a\subset C_b$. Hence $C_a=C_b$. Q.E.D.

Remark. From the argument above we see that if $b\in C_a$, then $C_b=C_a$. Thus any element in an equivalence class can be the representative.

Cosets

Definition. Let $G$ be a group and $H\le G$. Define a relation $\sim_L$ on $G$ by

Then $\sim_L$ is an equivalence relation. The group $G$ is then partitioned into equivalence classes, and each takes the form $aH$

We call these equivalence classes the left cosets of $H$ in $G$.

The collection of left cosets of $H$ in $G$ is denoted by $[G:H]$. The index $|G:H|$ of $H$ in $G$ is the size of $[G:H]$, namely, the number of cosets of $H$ in $G$.

Similarly, one can define $a\sim_Rb$ if and only if $b=ha$ for some $h\in H$. The corresponding equivalence classes take the form $Ha$.

Example. If $n\in\mathbb{Z}^+$, $[\mathbb{Z}:n\mathbb{Z}]={k+n\mathbb{Z}\mid k\in\mathbb{Z}}={n\mathbb{Z},\,1+n\mathbb{Z},\,…,\,(n-1)+n\mathbb{Z}}$.

Example. The Dihedral group $D_n={\text{id},r,…,r^{n-1},s,rs,…,r^{n-1}s}=\langle r\rangle\sqcup\langle r\rangle s$.

Example. Let $S_n$ be the $n$th symmetric group and $A_n$ the $n$th alternating group. Let $\tau$ be an odd permutation, say, $(1,2)$. Then $S_n=A_n\sqcup\tau A_n$.

Lagrange’s Theorem

Lemma. Every coset $aH$ has the same cardinality as $H$. If $|H|<\infty$, then $|aH|=|H|$.

Proof. Define $\phi:H\to aH$ by $\phi(h)=ah$. Then $\phi(H)=aH$ and thus $\phi$ is onto. Suppose $\phi(h_1)=\phi(h_2)$. Then $ah_1=ah_2$ and thus $h_1=h_2$. Hence $\phi$ is $1-1$. Therefore $\phi$ is bijective. Q.E.D.

Lagrange’s Theorem. Let $G$ be a finite group and let $H\le G$. Then $|G:H|=|G|/|H|$. Consequently, $|H|$ divides $|G|$.

Proof. $G$ is partitioned into the disjoint union of cosets of $H$:

(Since the cosets are disjoint and $G$ is finite, then there can only be finitely many distinct cosets of $G$.) Then $|G:H|=m$ and $|G|=|a_1H|+\cdots+|a_mH|=m|H|=|G:H|\cdot|H|$. Q.E.D.

Corollary. Number of left cosets of $H$ $=$ number of right cosets of $H$ $=$ $|G|/|H|$.

Corollary. If $G$ is a finite group and $g\in G$, then $|g|$ divides $|G|$. Thus $g^{|G|}=(g^{|g|})^{|G|/|g|}=e$.

Proof. Since $\langle g\rangle\le G$, then $|g|=|\langle g\rangle|$ divides $|G|$. Q.E.D.

Proposition. Every group of prime order is cyclic.

Proof. Suppose $p=|G|$ is a prime. $\forall\,g\in G\neq{e}$, $|\langle g\rangle|\ge 2$ divides $|G|=p$. Then $|\langle g\rangle|=p$ and hence $G=\langle g\rangle$. Q.E.D.
