Partition and Equivalence

S=⨆i∈ISiS=\bigsqcup_{i\in I}S_i S=i∈I⨆​Si​

where ⊔\sqcup⊔ represents disjoint union.

Definition. An equivalence relation on SSS is a relation “∼\sim∼” satisfying

  • (Reflexive/Reflexivity) a∼aa\sim aa∼a ∀ a∈S\forall\,a\in S∀a∈S
  • (Symmetric/Symmetry) a∼ba\sim ba∼b implies b∼ab\sim ab∼a
  • (Transitive/Transitivity) a∼ba\sim ba∼b and b∼cb\sim cb∼c imply a∼ca\sim ca∼c

Theorem.

  1. If {Si∣i∈I}\{S_i\mid i\in I\}{Si​∣i∈I} is a partition of SSS, then the relation ∼\sim∼ defined by "a∼ba\sim ba∼b if and only if a,b∈Sia,b\in S_ia,b∈Si​ for some iii" is an equivalence relation.
  2. If ∼\sim∼ is an equivalence relation on SSS, then {Ca∣a∈S}\{C_a\mid a\in S\}{Ca​∣a∈S}, where Ca={b∈S∣b∼a}C_a=\{b\in S\mid b\sim a\}Ca​={b∈S∣b∼a} is the equivalence class with representative aaa, is a partition of SSS.

Then ∀ a,b∈S\forall\,a,b\in S∀a,b∈S, CaC_aCa​ and CbC_bCb​ are either the same set or disjoint. If c∈Ca∩Cbc\in C_a\cap C_bc∈Ca​∩Cb​ for some c∈Sc\in Sc∈S, then c∼ac\sim ac∼a and c∼bc\sim bc∼b. Definition. Let GGG be a group and H≤GH\le GH≤G. Define a relation ∼L\sim_L∼L​ on GGG by

a∼Lb if and only if a−1b∈H\text{$a\sim_L b$ if and only if $a^{-1}b\in H$} a∼L​b if and only if a−1b∈H

HK={hk∣h∈H, k∈K}HK=\{hk\mid h\in H,\,k\in K\} HK={hk∣h∈H,k∈K}

By associativity of group operation, we have (HK)L=H(KL)(HK)L=H(KL)(HK)L=H(KL).

If H≤GH\le GH≤G, then HH=HHH=HHH=H. To see this, by closure we have HH⊂HHH\subset HHH⊂H, and H=eH⊂HHH=eH\subset HHH=eH⊂HH.
