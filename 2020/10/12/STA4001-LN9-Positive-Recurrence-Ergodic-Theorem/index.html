Theorem. Two communicating states are either both recurrent or both transient.

Theorem. If a finite DTMC is irreducible, then every state is recurrent.

Proof. If every state is transient, then each state is visited only a finitely many times. But then there are finitely many visits to all states. This is a contradiction.
